Description

This example gets the DPU associated with the stream given on the command line or piped/typed into STDIN. It presents it in a nice ASCII table.] Note that the CSDL must be enclosed in quotes if given on the command line.

ruby dpu.rb ‘interaction.content contains “football”’

or

cat football.csdl | ruby dpu.rb

NB: Most of the error handling (exception catching) has been removed for the sake of simplicity. Nearly everything in this library may throw exceptions, and production code should catch them. See the documentation for full details.
