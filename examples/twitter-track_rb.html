Description

This example constructs a DataSift_Definition object with CSDL that looks for anything containing the word “football”. It then gets an HTTP consumer for that definition and displays matching interactions to the screen as they come in. It will display 10 interactions and then stop.

NB: Most of the error handling (exception catching) has been removed for the sake of simplicity. Nearly everything in this library may throw exceptions, and production code should catch them. See the documentation for full details.
